## 🌟 프로젝트 배경 및 목적

대부분의 기존 지도 서비스는 최단 시간이나 최소 거리와 같은 '효율성'에 집중한 경로를 제공합니다.

하지만 보행자, 특히 아동, 노인, 여성 등 안전 취약계층에게는 '얼마나 빨리 가느냐'보다 '얼마나 안전하게 가느냐'가 더 중요한 지표입니다.

안심맵은 이러한 문제의식에서 출발하여 다음과 같은 구체적인 목적을 달성하고자 합니다.

* **사각지대 해소**: 단순히 밝은 길을 넘어, 실제 사고 이력과 범죄 발생 구역 데이터를 결합하여 사용자에게 실질적인 위험 정보를 제공합니다.
* **데이터 기반의 안전 판단**: "이 길 주변에 사고가 자주 일어나는가?", "이 동네의 치안 수준은 어떠한가?"와 같은 질문에 대해 주관적인 느낌이 아닌 공공데이터 기반의 수치로 답변합니다.
* **능동적 위험 회피**: 사용자가 경로를 탐색하는 단계에서 미리 위험 요소를 리스트화하여 보여줌으로써, 사고 발생 가능성을 사전에 인지하고 회피할 수 있도록 돕습니다.
